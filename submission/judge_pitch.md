# 评委视角亮点

## 一句话

城市踏勘官 CityScout 的核心叙事是“唤醒城市沉睡的基因”：它不是推荐去哪玩，而是调用高德地图能力，把城市里分散的地点、POI、路径和公共空间线索转化为可现场验证的证据路线。

## 5 个亮点

1. 故事母题清晰
   - 每座城市都有未被看见的线索：老街巷里的生活方式、被忽略的公共空间、断裂的慢行路径、尚未释放的商业活力。CityScout 把这些线索从地图里唤醒。

2. 真实痛点明确
   - 城市更新、商业选址、文旅策划都需要现场踏勘，但前期准备分散、耗时、容易漏点。

3. 高德能力不可替代
   - 地理编码定位故事起点，POI 搜索发现城市线索，路线规划串联现场证据，JSAPI 地图把证据可视化。

4. 与普通 Citywalk 明显不同
   - 普通 Citywalk 解决“去哪玩”，CityScout 解决“为了一个城市判断，去哪里看、看什么、拍什么、怎么汇报”。

5. Demo 可快速理解
   - 评委 1 分钟内能看到输入、真实高德地图、POI 点位、路线、点位表、风险机会和汇报摘要。

## README 首屏短文案

> CityScout 不是推荐你去哪里玩，而是帮你走进城市现场，唤醒那些沉睡在街巷、设施、路径和公共空间里的城市基因。它调用高德地理编码、POI 搜索、路线规划和 JSAPI 地图，把城市叙事落到真实地点和现场证据上。

## 报名表功能描述末尾

本作品的重点不是泛旅游推荐，而是把高德地图能力封装成一个城市现场证据 Skill：用地理编码定位故事起点，用 POI 搜索发现城市线索，用路线规划串联现场证据，用观察清单完成城市判断。
