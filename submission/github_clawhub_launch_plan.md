# GitHub / ClawHub 发布计划

目标：同时冲一等奖和 MVP 奖。一等奖看作品质量，MVP 奖与 ClawHub 下载量、GitHub 收藏量相关。

## GitHub 仓库

建议仓库名：

```text
city-scout-amap-skill
```

建议简介：

```text
高德开放平台 Skill：城市踏勘官 CityScout，唤醒城市更新、商业选址和文旅动线里的沉睡线索。
```

README 首屏必须包含：

- 一句话说明
- Demo 截图或 GIF
- 使用场景
- 高德能力调用链路
- 安装/使用方法
- ClawHub 下载链接
- Star 引导

## 发布后传播节奏

1. 先发 GitHub，确认 README、Demo、Skill 文件完整。
2. 再发 ClawHub，确保下载入口可用。
3. 发布外站文章，标题带“高德开放平台skill”。
4. 在文章里引导：
   - GitHub Star
   - ClawHub 下载
   - Demo 试用
