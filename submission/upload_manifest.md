# 上传材料清单

## 必传

1. 使用教程
   - 推荐上传：`submission/tutorial.md`
   - 如报名系统更偏好文档，可转成 PDF 后上传。

2. `skill.md` 文件
   - 推荐上传：`submission/skill.md`
   - 备用：`city-scout-amap-skill/SKILL.md`

## 强烈建议上传

3. Demo 演示
   - 推荐上传：录屏视频
   - 录屏脚本：`submission/demo_script.md`
   - Demo 文件：`demo/index.html`

4. GitHub / ClawHub 链接
   - 先发布 `city-scout-amap-skill/` 目录。
   - README 必须放 Demo 截图、ClawHub 下载入口和 Star 引导。

5. 外站传播链接
   - 推荐用：掘金 / CSDN / 公众号 / 小红书
   - 文案草稿：`submission/social_post.md`

## 不建议上传

- 不上传 `.env.example`，除非作为 GitHub 仓库说明文件。
- 不上传任何真实 API Key。
- 不上传未确认版权的图片素材。

## 打包建议

报名表上传时优先保持文件少而清晰：

- `skill.md`
- `tutorial.md`
- `demo_video.mp4`
- `demo/index.html` 或打包 zip

如果系统允许多个文件，可以补充：

- `README.md`
- `examples/example_prompt.md`
- `examples/example_output.md`
